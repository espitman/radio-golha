# Radio Golha Desktop (SwiftUI)

Modular macOS desktop app structure for Radio Golha.

Current implemented feature:
- `Features/Home`

Folder layout:
- `App/` app entry point and root composition
- `Core/` app-wide theme and foundation
- `Shared/` reusable cross-feature components
- `Features/` feature-first modules (Home, and future screens)

## Run
```bash
cd /Users/espitman/Documents/Projects/radioGolha/mac-desktop-app
swift run RadioGolhaDesktop
```
